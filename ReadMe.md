# Vim Color Schemes

All color schemes here support at least 256 colors.

How to use them:

1. Clone this repository, or just copy the files you need.
2. Put `*.vim` into `~/.vim/colors/`.
3. In Vim or `~/.vimrc`, try `:colorscheme moody` or `:colorscheme garden`.
